# Qgis-exercise-2
Geological map Camposauro, Campania, Italy
![digit geo camposauro](https://github.com/user-attachments/assets/a32b3d8b-a90e-4586-9694-87a0c28d32c6)
This geological map shows the different types of rock bodies in an area of ​​the province of Benevento in Campania, using colors and symbols that represent the formations based on their composition and structure. It also shows the various faults encountered and their types, the mountains, the location of the main inhabited areas, and the sampling points.
Coordinate reference system: EPSG:3004 Monte Mario/Italy zone 2 
Qgis version: QGIS 3.34 Prizren
